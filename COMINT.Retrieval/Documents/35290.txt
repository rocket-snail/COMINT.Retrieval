a directory of library instruction programs in indiana
    academic libraries. a survey of 40 academic libraries in
    indiana developed this directory of library user instruction
    programmes which provides an index profile of library resources
    and types of programmes within the state. entries are
    alphabetical in outline form and provide a library contact
    person, information about the programmes, and lists of topics
    for which print and audio-visual materials are available.
    report not available from ntis.
