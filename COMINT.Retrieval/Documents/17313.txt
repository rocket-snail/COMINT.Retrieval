a personal, computerised literature retrieval system. the
    computerised method of indexing and retrieving documents from a
    personal collection is described. consists of a univac 1108
    with input from cards and output via line printer. system
    allows input of author, title, bibliographic details and
    keywords while documents are allocated a numberic code and
    stored sequentially. 3 types of output are provided; alphabetic
    keywords list with document reference numbers, a reference list
    of documents titles, bibliographic details and authors, and
    output from searches on the system.
