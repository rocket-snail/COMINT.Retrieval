data retrieval systems and international publishing.
    describes problem areas in the economics of data base vending,
    bibliographic data base publishing and primary journal article
    publishing. discusses data base hosts; developments in the data
    base publishing industry; linking texts and graphics;
    integrated data base publishing; and research in document
    delivery, including a description of the adonis concept.
    concludes that a much closer relationship needs to be forged
    between data base publishers, librarians and end users to
    alert, at the earliest possible stage, any mismatch between
    application of technology and user acceptability. also stresses
    the need for useful and appropriate standards.
